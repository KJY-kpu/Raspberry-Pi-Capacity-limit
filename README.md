# Raspberry-Pi-Capacity-limit

## 수용인원제한 알림기

⁠코로나 팬데믹으로 인한 영업장의 인원 제한이 이루어지는 와중에


⁠무인으로 운영하는 영업장 혹은 일손이 부족한 상황에서 출입자의 수를

⁠파악하고 수용인원이 최대치에 달할 시 시.청각적으로 입장을 제한하는

⁠기능을 구현한다.
